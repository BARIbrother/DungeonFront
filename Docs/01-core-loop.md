# 코어 루프

> `00-vision.md`의 MVP 범위를 **플레이어가 실제로 겪는 흐름**으로 풀어 쓴 문서.  
> 기계·레시피·의뢰 상세는 아래 프레임워크에 채워 넣는다.

## MVP 전제

- **던전**: 뒷산 동굴 (기믹 없음)
- **시스템**: 하루 3단계, 의뢰, 납품, 골드, 명성, 기초 기계·레시피
- **제외**: `99-further_implementation.md` 항목 전부

---

## 하루 사이클 (프레임워크)

```
┌──────────┐    ┌──────────────┐    ┌──────────┐
│   준비   │ →  │ 생산 (5분)   │ →  │   결산   │ → 다음 일차
│ 탑다운   │    │ 탑다운       │    │ 별도 UI  │
└──────────┘    └──────────────┘    └──────────┘
```

### 준비 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | (필수) **의뢰 수락 클릭** | 스토리 의뢰는 반드시 수락 |
| 2 | **기계 배치·회수** | 인벤 ↔ 맵. 배치 모드: 하단 목록 → 클릭 → 맵 배치 |
| 3 | (선택) **기계 레시피 설정** | 우클릭 UI |
| 4 | (선택) **재료·기계·구역 구매** | 골드 |
| 5 | (선택) **기계·레시피 해금** | 명성 조건 충족 후 수동 해금 |
| 6 | (선택) **구역 확장** | 골드 |
| 7 | **생산 시작** | 화면 **중앙 상단** 버튼. 미배치·무의뢰 시 경고 |

### 생산 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | **기계 가동** | 틱 자동 + 제작기·마나 제작기 등 수작업 기계는 클릭 가공 |
| 2 | **물류** | 벨트·출력기·플레이어 수동 운반 |
| 3 | (선택) **레시피 전환** | 우클릭 UI |
| 4 | **고장 수리** | 클릭 (발생 시). **고장 알림** |
| 5 | **5분 경과** | 타이머 종료. **조기 종료 불가** |

**생산 종료 시 자동 처리**

- 기계 WIP → 재료 환원, 기계 상태 초기화
- 맵·기계에 남은 **완성품** → **인벤토리** 일괄 이동
- **요약 팝업** 표시

### 결산 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | **의뢰 목록 확인** | 납기·선택 시 **보유 n / 요구 m** |
| 2 | **물품 제출** | 버튼 클릭 → 인벤 자동 차감. **부분 납품 불가**. 부족 시 버튼 빨간색 + 안내 |
| 3 | **보상 수령** | 골드·명성 |
| 4 | (자동) **미납 처리** | 명성 차감 등 |
| 5 | **다음 일차** | 일차 +1 → 준비 단계 |

## 성장 루프 (프레임워크)

```
의뢰 수락 → 생산·납품 → 골드·명성
                ↓
    골드: 재료·기계·구역 확장
    명성: 기계 해금 → 상위 레시피·의뢰
                ↓
         더 어려운 의뢰 대응
```

## 기계 규칙

### 티어·타입

- 기계는 **티어**로 구분
- **가장 낮은 티어**의 이름이 해당 기계의 **type**(역할)과 같음  
  - 예: `수동 제작대` (티어1) → type `제작기`
- **같은 type**의 기계는 **같은 역할** (상위 티어는 처리 속도·가능 레시피 등에서 차별)

### 레시피 선택

- **준비 단계·생산 단계** 모두에서 기계 **우클릭** → UI에서 **작업 레시피 전환**
- 기계는 **현재 선택된 레시피만** 작업

## 기초 레시피 (MVP)

### 체인

```
철광석 노드 ─[채굴기]─→ 철 광석 ─[벨트/출력기/수동]─→ 용광로 ─→ 철 ─[벨트/출력기/수동]─→ 제작기 ─→ 철 막대 / 철 판
```

- **출력기**: 뒤쪽 기계 출력 포트 → 앞 칸 운반. 없으면 플레이어 수동 운반
- **벨트**: Factorio형 이송 (MVP 포함)

### 레시피 목록

| 출력 | 입력 | type | 소요 | 비고 |
|------|------|------|------|------|
| **철 광석** | *(없음)* | `채굴기` | **30틱** | 맵의 **철광석 노드**에서 채취 |
| **철** | 철 광석 ×1 | `용광로` | **15틱** | |
| **철 막대** | 철 ×1 | `제작기` | **5회 클릭** | 수작업 |
| **철 판** | 철 ×1 | `제작기` | **5회 클릭** | 수작업, 제작기에서 레시피 전환 |

- 제작기는 **철 막대** / **철 판** 중 UI로 선택 후 작업
- 채굴기·용광로도 해당 type이 처리 가능한 레시피 중 선택 (MVP에서는 위 항목 각 1종)

### 틱 참고 (자동 공정)

| 레시피 | 틱 | 실시간 |
|--------|-----|--------|
| 철 광석 | 30 | 3.0초/개 |
| 철 | 15 | 1.5초/개 |

## 의뢰 (MVP)

### ID 규칙

- **8자리 십진수**
- **앞 3자리**: 던전 번호 (예: `001` = 뒷산 동굴)
- **뒤 5자리**: 의뢰 번호

```
001 00001  →  00100001
└─┘ └────┘
던전  의뢰
```

### 경제 수치

- **골드**·**명성**의 공급·소비량: *미정* (의뢰 보상·구매·페널티 등)

### 의뢰 목록

| ID | 던전 | 종류 | 요구 물품 | 납기 | 보상 | 비고 |
|----|------|------|-----------|------|------|------|
| `00100001` | 001 뒷산 동굴 | 튜토리얼 (필수) | 철 광석 ×5, 철 ×5, 철 판 ×1, 철 막대 ×1 | **당일** | 채굴기 ×1, 골드, 명성 *(량 미정)* | 아이템 보상 없음 |
| *(예시)* | 001 | 일반 | 철 판 ×500, 기사단 맹세의 검 ×5 | **5일** | *(미정)* | 멀티데이 예시 |

**00100001 상세**

| 항목 | 내용 |
|------|------|
| 요구 | 철 광석 5, 철 5, 철 판 1, 철 막대 1 |
| 보상 | 채굴기 1대, 골드 *(량 미정)*, 명성 *(량 미정)* — **이 세 가지 외 보상 없음** |
| 수락 | **반드시 수락 클릭** |

## 스토리·튜토리얼 (확정)

### 스토리 개요

- 주인공은 대장간을 물려받았으나 **정식 대장장이 면허 없음**
- **면허 시험 통과**가 목표
- 이벤트: **오프닝** · **필수 의뢰 전후** · **클리어**

### 주요 NPC

| 이름 | 역할 | 등장 |
|------|------|------|
| **이브** | 대장간 협회 자격 평가원. 엔딩 필수 의뢰 전달 | 1일차 |
| **레이** | 대장간 직원, 마법 테크 | 3일차 (스토리 only: 1일차) |

### 첫 플레이 튜토리얼 순서

1. **(준비)** 의뢰·수락 방법
2. **(생산)** 채굴기 배치 → 용광로 배치 → 채굴기–용광로 간 **수동 운반** → 창고(인벤)로 수동 이동
3. **(결산)** 요구 물품 납품
4. **(다음 준비)** 명성으로 기계 해금 → 골드로 방금 열린 기계 구매

- 튜토 **대상 강조**: 화살표·하이라이트·카메라 이동 **전부** 사용
- 스토리·튜토 **스킵**: ESC → 확인 팝업
- 재생 중 **게임 일시정지**
