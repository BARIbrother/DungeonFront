# 코어 루프

> `00-vision.md`의 MVP 범위를 **플레이어가 실제로 겪는 흐름**으로 풀어 쓴 문서.  
> 기계·레시피·의뢰 상세는 아래 프레임워크에 채워 넣는다.

## MVP 전제

- **던전**: 뒷산 동굴 (기믹 없음)
- **시스템**: 하루 3단계, 의뢰, 납품, 골드, 명성, 기초 기계·레시피
- **제외**: `99-further_implementation.md` 항목 전부

---

## 하루 사이클 (프레임워크)

```
┌──────────┐    ┌──────────────┐    ┌──────────┐
│   준비   │ →  │ 생산 (5분)   │ →  │   결산   │ → 다음 일차
│ 탑다운   │    │ 탑다운       │    │ 별도 UI  │
└──────────┘    └──────────────┘    └──────────┘
```

### 준비 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | (선택) **의뢰 확인·수락** | 하루 최대 3개, 0개 가능 |
| 2 | **기계 배치·회수** | 인벤 ↔ 맵, 제한 없음 |
| 3 | (선택) **기계 레시피 설정** | 우클릭 UI |
| 3 | (선택) **재료 구매** | 골드로 일시 구매 |
| 4 | (선택) **기계 구매·해금** | 명성 조건 충족 시 |
| 5 | (선택) **구역 확장** | 골드로 구역 구매 |
| 6 | **생산 시작** | 준비 종료 → 생산 단계 진입 |

### 생산 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | **기계 가동** | 틱 자동 + 모루는 클릭 가공 |
| 2 | (선택) **레시피 전환** | 우클릭 UI |
| 3 | **고장 수리** | 클릭 (발생 시) |
| 4 | **5분 경과** | 타이머 종료 시 생산 단계 종료 |

**생산 종료 시 자동 처리**

- 기계 WIP → 재료 환원, 기계 상태 초기화
- 완성품 → 인벤토리

### 결산 단계 — 플레이어가 하는 일

| 순서 | 행동 | 비고 |
|------|------|------|
| 1 | **의뢰 목록 확인** | 납기·진행도 |
| 2 | **물품 제출** | 버튼 클릭 → 인벤 자동 차감 |
| 3 | **보상 수령** | 골드·명성 |
| 4 | (자동) **미납 처리** | 명성 차감 등 |
| 5 | **다음 일차** | 일차 +1 → 준비 단계 |

## 성장 루프 (프레임워크)

```
의뢰 수락 → 생산·납품 → 골드·명성
                ↓
    골드: 재료·기계·구역 확장
    명성: 기계 해금 → 상위 레시피·의뢰
                ↓
         더 어려운 의뢰 대응
```

---

## 생산 단계 밸런싱

| 항목 | 값 |
|------|-----|
| 생산 단계 길이 | **5분** (300초) |
| 틱 환산 | **1초 = 10틱** |
| 생산 단계 총 틱 | **3000틱** |

- 자동 가공 레시피의 소요 시간은 **틱** 단위로 표기
- 수작업(모루 등)은 **클릭 횟수**로 별도 표기

## 시작 시 기본 지급

| 기계 | 수량 | type | 비고 |
|------|------|------|------|
| 채굴기 | 1 | `채굴기` | 최저 티어 = 타입명 |
| 용광로 | 1 | `용광로` | |
| 수동 모루 | 1 | `모루` | 최저 티어, type는 `모루` |

## 기계 규칙

### 티어·타입

- 기계는 **티어**로 구분
- **가장 낮은 티어**의 이름이 해당 기계의 **type**(역할)과 같음  
  - 예: `수동 모루` (티어1) → type `모루`
- **같은 type**의 기계는 **같은 역할** (상위 티어는 처리 속도·가능 레시피 등에서 차별)

### 레시피 선택

- **준비 단계·생산 단계** 모두에서 기계 **우클릭** → UI에서 **작업 레시피 전환**
- 기계는 **현재 선택된 레시피만** 작업

---

## 기초 기계 (MVP)

| type | 최저 티어 이름 | 역할 | 수작업 | 시작 지급 |
|------|----------------|------|--------|-----------|
| `채굴기` | 채굴기 | 자원 노드 채취 | 없음 (틱) | ○ |
| `용광로` | 용광로 | 광석·재료 제련 | 없음 (틱) | ○ |
| `모루` | 수동 모루 | 금속 가공·성형 | **클릭** | ○ |

---

## 기초 레시피 (MVP)

### 체인

```
철광석 노드 ─[type:채굴기]─→ 철 광석 ─[type:용광로]─→ 철 ─[type:모루]─→ 철 막대 / 철 판
```

### 레시피 목록

| 출력 | 입력 | type | 소요 | 비고 |
|------|------|------|------|------|
| **철 광석** | *(없음)* | `채굴기` | **30틱** | 맵의 **철광석 노드**에서 채취 |
| **철** | 철 광석 ×1 | `용광로` | **15틱** | |
| **철 막대** | 철 ×1 | `모루` | **5회 클릭** | 수작업 |
| **철 판** | 철 ×1 | `모루` | **5회 클릭** | 수작업, 모루에서 레시피 전환 |

- 모루는 **철 막대** / **철 판** 중 UI로 선택 후 작업
- 채굴기·용광로도 해당 type이 처리 가능한 레시피 중 선택 (MVP에서는 위 항목 각 1종)

### 틱 참고 (자동 공정)

| 레시피 | 틱 | 실시간 |
|--------|-----|--------|
| 철 광석 | 30 | 3.0초/개 |
| 철 | 15 | 1.5초/개 |

## 의뢰 (MVP)

### ID 규칙

- **8자리 십진수**
- **앞 3자리**: 던전 번호 (예: `001` = 뒷산 동굴)
- **뒤 5자리**: 의뢰 번호

```
001 00001  →  00100001
└─┘ └────┘
던전  의뢰
```

### 경제 수치

- **골드**·**명성**의 공급·소비량: *미정* (의뢰 보상·구매·페널티 등)

### 의뢰 목록

| ID | 던전 | 종류 | 요구 물품 | 납기 | 보상 | 비고 |
|----|------|------|-----------|------|------|------|
| `00100001` | 001 뒷산 동굴 | 튜토리얼 (필수) | 철 광석 ×5, 철 ×5, 철 판 ×1, 철 막대 ×1 | *미정* | 채굴기 ×1, 골드, 명성 | 수치 미정 |

**00100001 상세**

| 항목 | 내용 |
|------|------|
| 요구 | 철 광석 5, 철 5, 철 판 1, 철 막대 1 |
| 보상 | 채굴기 1대, 골드 *(량 미정)*, 명성 *(량 미정)* |
