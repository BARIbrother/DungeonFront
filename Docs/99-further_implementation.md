## 던전 복수화

**요약**: 던전마다 별도 대장간(공장)을 두되, **일차·인벤토리는 공유**. 한 던전에서 막히면 다른 던전에서 생산·테크를 쌓아 돌파하는 플레이.

- 현재 `00-vision.md` 기준: 던전은 배경·분위기용 (단일 던전 앞 대장간)
- 본 항목: **던전을 여러 개** 두고, 매 일차마다 어느 던전에서 플레이할지 선택

### 플레이 방식

- **매 일차(하루) 시작 시** 진행할 **던전을 선택**
- 선택한 던전의 대장간에서 그날의 준비 → 생산 → 결산 진행 (`00-vision.md` 하루 구조 동일)

### 공유 vs 비공유

| 공유 (전 던전 공통) | 비공유 (던전별 독립) |
|---------------------|----------------------|
| **일차** (게임 진행 일수) | **현재 배치된 기계** (맵 위 설비·레이아웃) |
| **인벤토리** (완성품·재료·**기계** 등) | **의뢰 목록** |
| | **명성** |

### 설계 의도 (크로스 던전 플레이)

```
던전 A에서 의뢰 막힘
    ↓
던전 B 선택 → B에서 명성·테크 쌓고, 상위 기계 제작
    ↓
완성품·기계를 공유 인벤토리에 보관
    ↓
다시 던전 A 선택 → 인벤의 물품·기계로 A 공장 보강·의뢰 클리어
```

- 한 던전에서 진행이 막혔을 때, **다른 던전에서 테크를 열고 기계를 만든 뒤** **공유 인벤으로 가져와** 막힌 던전을 돌파할 수 있음
- **기계도 인벤토리에 보관** → 던전 간 이동·재배치 가능
- **준비 단계**에서 인벤 기계 **자유 배치·회수** (조건·비용 없음, `00-vision.md`와 동일)
- 맵에 **배치된** 상태와 인벤 **보관** 상태는 구분 (배치·명성·의뢰는 던전별)
- 인벤·일차 공유 → 던전 간 **물품·기계 이동**으로 협력·우회 클리어

### 던전 목록 (6종)

| # | 이름 | 역할 | 핵심 기믹 |
|---|------|------|-----------|
| 1 | **뒷산 동굴** | 튜토리얼·입문 | 기믹 없음, 기본 던전 |
| 2 | **혹한의 설원** | 환경 제약 | 냉기 수치, 열원 인접 필요, 결빙·수동 해빙 |
| 3 | **슬라임 서식지** | 단일 원료 가공 | 자원은 `비정형 슬라임` 1종만, 다단 가공으로 모든 생산품 |
| 4 | **깊이 잠든 황금향** | 경제 특수 | **명성 없음**, 골드(PTW)로 모든 것 해결 |
| 5 | **좋은 대장간?** | 의뢰 해석 | 애매한·부정확한·상성 안 맞는 의뢰, 재량 플레이 |
| 6 | **태고의 땅** | 생물 공정 | 포획·사육·생물을 생산 매개로 사용 |

---

#### 1. 뒷산 동굴

- **아무 기믹도 없는 기본 던전**
- 플레이어가 **처음 접하는** 던전
- `00-vision.md`의 표준 룰 그대로 익히는 공간

#### 2. 혹한의 설원

- **냉기 수치** 추가
- 기계는 **열원 기계에 인접**하지 않으면 서서히 **얼어붙음**
- 얼어붙은 기계는 플레이어가 **직접 얼음을 깨** 줘야 다시 가동

#### 3. 슬라임 서식지

- 채취 가능한 자원은 **`비정형 슬라임` 단 1종**뿐
- 슬라임을 **여러 장치**에 거쳐 가공해 원하는 생산품 출력
- 극단적 **단일 원료 → 다품종** 퍼즐

#### 4. 깊이 잠든 황금향

- **명성 시스템 없음** (던전별 명성 규칙의 예외)
- 모든 것을 **골드로 해결**하는 **PTW** 구조
  - 고장 수리, 납품 의뢰 미달, 기타 복잡한 공정 등 **골드 지불로 해결 가능**
- 다른 던전에서 쌓은 명성·테크와 별개의 **순수 경제** 던전

#### 5. 좋은 대장간?

- **컨셉**: 집에서 놀기만 하는 귀족 자제를 위해, 부모가 돈 들여 조성한 **가짜 던전**
- 의뢰가 다른 던전과 다름:
  - **애매모호**한 요구
  - **정확한 수량**을 주지 않음
  - 정확히 요구해도 **상대 몬스터와 상성이 안 맞는 장비**를 요구하기도 함
- **목표**: 진상(?) 요구에서 **명시되지 않은 부분에 재량**을 발휘해 모험가들을 **무사 귀환**시키는 것
- 클리어 기준: 단순 납품 수량 충족이 아닌 **해석·판단** 중심
- **무사 귀환 판정**: 별도 **알고리즘**으로 구현 예정 *(상세 추후)*

#### 6. 태고의 땅

- **생물**을 포획·사육하고 **생산 공정에 투입**하는 컨셉
- 예시:
  - **용암 와이번** 포획 → 내뿜는 불꽃으로 **최고 티어 금속 제련**
  - **라플레시아-베타** → 장비에 포션을 먹여 **버프 효과를 장비에 부여**하는 매개

### 스토리·진행

- **메인 스토리**는 **필수 의뢰(메인 임무)** 완수로 진행
- 메인 임무를 클리어할 때마다 안내 표시:
  - *「다음 메인 임무는 ○○ 던전에서 진행합니다」*
- 플레이어에게 **다음에 어느 던전으로 갈지** 스토리가 직접 가이드
- 일차 시작 시 던전 선택은 자유이나, 메인 임무는 해당 던전에서 수행

### 미정·추후 정리

- 던전 **언락 순서** *(미정)*
- 던전 선택 UI
- 던전 5 무사 귀환 **알고리즘** 상세
