## Story/ 폴더 규칙

| 항목 | 규칙 |
|------|------|
| 파일명 | `{대표 eventId}-{slug}.md` (예: `001E00001-Start-Tutorial.md`) |
| 상단 | **의뢰** 표 + **포함 이벤트** 목록 |
| 본문 | 이벤트별 — 메타 표 → **대사** 표 → (튜토 시) **튜토 강조** 표 |
| 대사 key | `{eventId}_{순번}` (예: `001E00002_01`) |
| 새 파일 | 의뢰·이벤트 확정 후 `04-story` 표에 **상세** 링크 추가 |
