# 개발 계획 (7주)

> **기간**: 7주 (핵심 개발) · 여유 2주 가능 시 8~9주차 폴리시·QA  
> **MVP 범위**: `../00-vision.md` · `../01-core-loop.md` · `../02-data-structure.md`  
> **현재 코드**: `PlayerMovement.cs` 스켈레톤만 존재

---

## MVP 완료 기준

| # | 항목 | 완료 조건 |
|---|------|-----------|
| 1 | 하루 3단계 | 준비 → 생산(5분) → 결산이 끊김 없이 1회 이상 순환 |
| 2 | 생산 | **10종 이상의 기계**·**벨트**·**출력기**·물류로 레시피 제작 가능 |
| 3 | 의뢰 | 수락(하루 최대 3) · 당일/멀티데이 납기 · 결산 납품 · 보상(골드·명성) |
| 4 | 필수 의뢰 | 실패 시 게임오버 → **다른 세이브**로 재도전 (autosave) |
| 5 | 경제 | 골드(재료·**기계**·구역 구매·제작), 명성(**해금**·의뢰 풀) |
| 6 | 세이브 | JSON 저장·불러오기 (일차·인벤·배치·의뢰 진행) |
| 7 | 콘텐츠 | 뒷산 동굴 튜토리얼~클리어까지 의뢰·**스토리** 라인 플레이 가능 |

**이번 범위 밖**: 던전 복수화, 특수 기믹 던전, 엔드 **이후** 확장 스토리 (`../99-further_implementation.md`)

> **10종 기계**: 서로 다른 기계 **종류(type)** 10개 이상. 플레이어가 해금·배치·가동할 수 있어야 함.

## 개발

게임 로직 전반 — 하루 3단계, 공장 맵·생산, 의뢰·경제, **스토리·튜토리얼 연출**, UI, 세이브. 코드는 `Assets/Scripts/`, 데이터 구조는 `../02-data-structure.md` 기준.

### 코어·하루 구조

- [ ] 게임 세션 상태 (일차·골드·명성·인벤·공장·의뢰)
- [ ] **페이즈 전환** (준비 → 생산 → 결산 → 다음 날)
- [ ] 일차 증가·HUD 표시
- [ ] 새 게임 시작·초기 지급 (인벤 기계 4종)
- [ ] **스토리 only** 모드 분기

### 공장 맵·배치

- [ ] **그리드 맵**·타일·좌표 체계 (월드 ↔ grid)
- [ ] 탑다운 Factory 씬·카메라
- [ ] **배치 모드** UI (하단 인벤 기계 목록 → 선택 → 맵 클릭)
- [ ] **기계 배치·회수** (인벤 ↔ 맵, 준비만). 충돌·구역·노드 규칙
- [ ] **자원 노드** — 채굴기는 노드 위 1대
- [ ] **구역 확장** 시 타일·노드 추가
- [ ] 기계·노드 **Prefab** (종류별 점유 크기 상이)

### 플레이어·생산

- [ ] 플레이어 **WASD 이동** (탑다운)
- [ ] **생산 틱** (10 tick/s, 생산 단계 3000 tick = 5분)
- [ ] 기계 **공통 생산 프레임워크** (자동 틱 / 수작업 클릭 분기)
- [ ] 기계별 **WIP**·레시피 처리 (10종+ — SO 추가로 확장)
- [ ] **수작업 기계** (클릭 N회 가공, 제작기 등)
- [ ] 기계 **우클릭 레시피 전환** UI
- [ ] 생산 종료 시 WIP → 재료 환원, 완성품 → 인벤
- [ ] **고장** 발생·클릭 수리
- [ ] 플레이어 **상호작용** (수리·수작업·**수동 운반**)

### 물류 (벨트·출력기)

- [ ] **컨베이어 벨트** 배치·이송 (Factorio형)
- [ ] **출력기** — 뒤쪽 출력 포트 → 앞 칸 운반
- [ ] 기계 **입·출력 포트** (방향 자유)
- [ ] 생산 중 **수동 회수** · 플레이어 **직접 운반**
- [ ] 생산 종료 **요약 팝업** + 완성품 인벤 일괄

### 인벤토리·경제

- [ ] 인벤토리 (아이템 무제한·기계 별도 목록)
- [ ] 골드·명성 증감
- [ ] 준비 단계 **재료·기계·구역 골드 구매**
- [ ] **명성 기반 해금** (기계·레시피 수동 해금)
- [ ] **의뢰 출현 풀** (누적 명성 threshold)
- [ ] 해금·구매·**제작** UI

### 의뢰·결산

- [ ] 의뢰 수락 (하루 최대 3개)
- [ ] 준비 단계 **의뢰 목록·수락 UI**
- [ ] **결산 화면** (별도 Scene 또는 UI)
- [ ] **납품** — 인벤 자동 차감·요구 충족 판정
- [ ] **보상** 지급 (골드·명성·아이템·기계)

- [ ] **필수 의뢰 실패** → 게임오버 → **다른 세이브** 재도전

- [ ] **멀티데이** 납기 추적

### 스토리·튜토리얼 연출

- [ ] **대화 UI** + **엔딩**(대화만)

- [ ] **타이틀**·슬롯·새 게임/불러오기
- [ ] **설정** (볼륨·해상도·창 모드)
