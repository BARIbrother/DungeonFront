# 경제 수치 lock

**역할**: Dev2 · **Week**: 4 · **Issue**: 01  
**선행**: W3 경제·해금 UI

## 1. 이 작업물

뒷산에서 쓰는 골드 단가·구역 비용·명성 해금을 표로 고정하고 SO/테이블에 반영한다.

## 2. 확정 규칙

| 항목 | 내용 |
|------|------|
| 하루 최대 수락 | 3 (0 OK · 필수는 반드시 클릭) |
| 일반 미납 | 보상 명성의 50% 차감 |
| 필수 미납 | 게임오버 → 다른 세이브 |
| 부분 납품 | 불가 · 부족 시 버튼 빨강 |
| 골드 용도 | 재료·기계·구역·일부 제작 비용 |
| 명성 용도 | 기계/레시피 수동 해금 · 의뢰 풀 자동 해금 |

NewGame: 골드 0 · 명성 0.

## 3. 해금 (확정·초안)

| 조건 | 해금 |
|------|------|
| 시작 | `채굴기_1` · `용광로_1` · `제작기_1` · `창고_1` |
| 2번째 스토리 의뢰 클리어 | 마나 추출기·마법 부여대(보상) · 마나 제작기·마나 저장소(해금) |
| 명성 350 | `제단_1` |
| 명성 500 | `주조소_1` |
