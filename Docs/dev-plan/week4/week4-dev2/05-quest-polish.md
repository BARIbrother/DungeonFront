# 결산·의뢰 폴리시

**역할**: Dev2 · **Week**: 4 · **Issue**: 05  
**선행**: W3 결산·게임오버 · 뒷산 의뢰·클리어 라인·일반 풀

## 1. 이 작업물

결산·미납·게임오버·멀티데이가 뒷산 클리어 플로우에서 깨지지 않게 폴리시한다.

## 2. 결산 규칙

| 항목 | 내용 |
|------|------|
| 표시 | 보유 n / 요구 m |
| 납품 | 버튼 클릭 → 인벤 차감 · 부분 납품 불가 |
| 부족 | 버튼 빨강 + 안내 |
| 보상 | 골드·명성 |
| 일반 미납 | 명성 50% 차감 |
| 필수 미납 | 게임오버 → 다른 슬롯 |
| 멀티데이 | D-n 표시 · 일차 감소 |

## 3. 완료 기준

- [ ] 결산 n/m · 부분 납품 불가 · 버튼 상태
- [ ] 멀티데이 D-n 표시·일차 감소
- [ ] 필수 미납 → 게임오버 → 다른 슬롯
