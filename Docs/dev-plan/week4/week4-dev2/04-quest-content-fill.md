# 일반 풀·보상 폴리시

**역할**: Dev2 · **Week**: 4 · **Issue**: 04  
**선행**: 뒷산 의뢰 채우기 · 경제 수치 lock

## 1. 이 작업물

뒷산 일반 의뢰 풀 밀도·보상·명성 해금 연동을 플레이 가능 수준으로 맞춘다.  
필수 라인을 방해하지 않는 선에서 골드·명성 수급을 안정화한다.

## 2. 규칙

| 항목 | 내용 |
|------|------|
| 하루 최대 수락 | 3 |
| 일반 미납 | 보상 명성의 50% |
| 풀 해금 | 명성 threshold (초안: 0 / 50 / 150) |
| UI | 필수와 일반 구분 |

## 3. 작업 요약

명성 구간별 풀이 비지 않게 · 보상으로 구역·기계 구매 대략 가능 · 필수/일반 UI 구분.

## 4. 완료 기준

- [ ] 명성 구간별 풀에 의뢰가 비지 않음
- [ ] 하루 최대 3 수락 회귀
- [ ] 필수와 일반이 UI에서 구분
- [ ] 보상으로 구역·기계 구매 대략 가능
- [ ] 엔딩까지 경제 soft-lock 없음 · 경제 lock 표와 일치
