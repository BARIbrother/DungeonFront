# 클리어 라인 lock

**역할**: Dev2 · **Week**: 4 · **Issue**: 03  
**선행**: 뒷산 의뢰 채우기

## 1. 이 작업물

뒷산 클리어 라인을 lock한다.  
순서·해금 데이터는 **Drive → Utility program** 폴더의 퀘스트 라인 `.html`에 있다.

**코드**: `QuestManager` — 메인 진행·동시 오픈 · 클리어 시 엔딩 (`OnBackCaveCleared` / `StoryEventBus`)

## 2. 오픈 규칙

- **메인 의뢰**가 진행 기준이다.
- html에서 메인 의뢰와 **같은 행**에 있는 의뢰들은, 그 메인 의뢰가 열릴 때 **동시에** 열린다.

## 3. 클리어·엔딩

| 항목 | 내용 |
|------|------|
| 목표 | 대장장이 면허 취득 |
| 클리어 | 메인(스토리) 라인 완료 → 스토리 엔딩 |
| 필수 미납 | 게임오버 → 다른 세이브 (W3 회귀) |

## 4. 완료 기준

- [ ] html 기준 메인 의뢰 순서 lock
- [ ] 같은 행 의뢰가 메인과 동시 오픈
- [ ] 메인 라인 클리어 시에만 엔딩 트리거 1회
- [ ] 필수 미납 → 게임오버 회귀
