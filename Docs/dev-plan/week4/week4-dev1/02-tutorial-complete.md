# 튜토리얼 완성

**역할**: Dev1 · **Week**: 4 · **Issue**: 02  
**선행**: W3 튜토리얼 패널(최소 2~3단계) · UI 폴리시

## 1. 이 작업물

뒷산 첫 플레이 튜토리얼을 기획 순서대로 완주 가능한 세트로 채운다.  
강조·일시정지·스킵이 실제 클리어 플로우와 맞아야 한다.

**코드**: `TutorialPanelUI`, `TutorialHighlighter`, `TutorialState`

## 2. 확정 규칙

- 대상 강조: 화살표 · 하이라이트 · 카메라 이동 **전부**
- 스킵: ESC → 확인 팝업
- 재생 중 게임 일시정지
- NewGame: 골드 0 · 명성 0 · 맵 중앙 스폰 · 오프닝 후 바로 Prepare

## 3. 튜토 순서 (확정)

1. **(Prepare)** 의뢰·수락 방법
2. **(Production)** 채굴기 배치 → 용광로 배치 → 채굴기–용광로 **수동 운반** → 창고(인벤)로 수동 이동
3. **(Settlement)** 요구 물품 납품
4. **(다음 Prepare)** 명성으로 기계 해금 → 골드로 방금 열린 기계 구매

## 4. 1일차 이벤트·의뢰

| 순서 | eventId | 요약 |
|------|---------|------|
| 1 | `001E00001` | 오프닝 (면허·실기 시험) |
| 2 | `001E00002` | 이브 — 첫 의뢰 안내 |
| 3~ | `001E00003`~ | 튜토 (상세 미정) |

**필수 의뢰 `00100001`**

| 항목 | 내용 |
|------|------|
| 요구 | `iron_ore`×5, `iron`×5, `iron_plate`×1, `iron_rod`×1 |
| 납기 | 당일 |
| 보상 | `채굴기_1`×1, 골드 *(량 미정)*, 명성 *(량 미정)* |
| 수락 | 반드시 수락 클릭 |

시작 인벤: `채굴기_1` · `용광로_1` · `제작기_1` · `창고_1` 각 1.

NPC: `eve`(이브) 1일차.

단계별 확정 문구가 필요하면 `02-tutorial-complete_missingReq.md`를 채운다.
