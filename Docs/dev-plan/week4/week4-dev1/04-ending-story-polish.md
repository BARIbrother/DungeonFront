# 엔딩·스토리 폴리시

**역할**: Dev1 · **Week**: 4 · **Issue**: 04  
**선행**: W3 대화·스토리 · 튜토리얼 완성

## 1. 이 작업물

뒷산 MVP 스토리 엔딩(대장장이 면허 취득) 연출을 완성한다.

**코드**: `DialogueUI`, `EndingFlow` (신규 가능)

## 2. 클리어·엔딩

| 항목 | 내용 |
|------|------|
| 목표 | 대장장이 면허 취득 |
| 클리어 | MVP 모든 스토리 의뢰 완료 → 스토리 엔딩(대화) |
| 트리거 | 마지막 필수 의뢰 납품 |
| 엔딩 요약 | 면허 취득 · MVP 클리어 |
| 스킵 | ESC → 확인 팝업 · 재생 중 일시정지 |

## 3. 흐름

```
뒷산 필수 의뢰 전부 클리어
  → StoryEventBus Ending
  → Dialogue (면허 취득)
  → Title 또는 Continue(자유 플레이)
```

주요 NPC: 이브(`eve`) · 레이(`ray`). 레이는 일반 3일차 / 스토리 only 1일차.

엔딩 대사·Title vs Continue가 미정이면 `04-ending-story-polish_missingReq.md`를 채운다.
