# 세이브 안정화

**역할**: Dev1 · **Week**: 4 · **Issue**: 03  
**선행**: W3 세이브/로드

## 1. 이 작업물

뒷산 단일 던전 세이브·로드를 튜토~엔딩 완주 기준으로 안정화한다.

**코드**: `SaveData`, `SaveLoadService`

## 2. 규칙

| 항목 | 내용 |
|------|------|
| 저장 | 자동 + 수동 |
| autosave | 일차·필수 의뢰 클리어 시 (결산 후 등 시점 정리) |
| 필수 실패 | 다른 세이브로 재도전 |
| round-trip | 일차·인벤·골드·명성·factory·quests |
| factory | 배치·WIP·구역 해금(`unlockedZones`) |
| quests | 진행 중·멀티데이 |

## 3. 복원 대상

배치·WIP·구역 해금 · 진행 중 의뢰·멀티데이 · Title 로드 시 동일 페이즈/일차 · 게임오버 후 다른 슬롯 · version 호환.

## 4. 완료 기준

- [ ] 튜토 직후 세이브 → 로드 → 진행 계속
- [ ] 배치·WIP·구역·의뢰(멀티데이) round-trip
- [ ] Title 로드 → 동일 페이즈/일차
- [ ] 클리어 직전 세이브 → 로드 → 엔딩 가능
- [ ] Corrupt/부분 null 시 크래시 없이 안내
