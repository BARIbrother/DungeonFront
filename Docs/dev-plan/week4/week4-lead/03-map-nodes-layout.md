# 맵·자원 노드

**역할**: Lead · **Week**: 4 · **Issue**: 03  
**선행**: 구역 확장 · `MinerMachine`

## 1. 이 작업물

뒷산 Factory 맵의 시작 레이아웃·철광석(및 MVP 자원) 노드 위치·개수를 확정·배치한다.  
튜토 첫날과 클리어까지 채굴 병목이 나지 않게 한다.

**코드/씬**: Factory 씬 · 자원 노드 Prefab · (선택) `ResourceNode`

## 2. 배치 초안

| 구역 | 노드 | 개수 (초안) | 비고 |
|------|------|-------------|------|
| `zone_start` | 철광석 | 2~3 | 튜토 · `00100001` |
| `zone_east_1` | 철광석 또는 상위 | 1+ | 확장 보상감 |
| (선택) | 마나 원석 등 | 스토리 해금 후 | 마법 라인 |

`00100001` 요구: `iron_ore`×5, `iron`×5, `iron_plate`×1, `iron_rod`×1 (당일).  
채굴기: 자원 노드 위 1대만.

좌표·최종 개수가 미정이면 `03-map-nodes-layout_missingReq.md`를 채운다.

## 3. 구현 요약

노드 Prefab·채굴기 footprint 정렬 · 시작 맵 배치 · 확장 구역 해금 시 노드 활성/스폰 · 최종 좌표·개수는 씬(정본) 또는 문서 기록.

## 4. 완료 기준

- [ ] 노드 Prefab이 채굴기 footprint와 정렬
- [ ] NewGame 직후 철광석 채굴 가능 (치트 없이)
- [ ] 확장 구역 해금 시 노드 활성/스폰
- [ ] 튜토 의뢰량을 5분×수일 내 생산 가능 (밸런스 대략)
- [ ] Art 타일과 충돌·구멍 없음
