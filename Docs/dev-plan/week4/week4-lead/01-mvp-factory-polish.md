# 공장 폴리시

**역할**: Lead · **Week**: 4 · **Issue**: 01  
**선행**: Week 3 MVP 통합 (10종+ · 수작업/운반 · 레시피 lock)

## 1. 이 작업물

뒷산 Factory의 물류·생산·고장 버그와 갭을 막아, 튜토~엔딩 완주에 지장 없게 한다.

**코드**: `Assets/Scripts/Machine/`, `Assets/Scripts/Placement/`, Factory 씬

## 2. 작업 내용

- 벨트·출력기·수동 회수 우선순위·버그
- 입·출력 포트·WIP 회귀
- 고장 tint·수리 클릭 수 조정
- Week 3 이슈 triage

## 3. 생산 상수 (회귀 기준)

| 항목 | 값 |
|------|-----|
| 생산 단계 길이 | 5분 (300초) |
| 틱 | 1초 = 10틱 · 총 3000틱 |
| 조기 종료 | 불가 |

생산 종료 시: WIP → 재료 환원 · 기계 상태 초기화 · 맵·기계 완성품 → 인벤 일괄 · 요약 팝업.

## 4. 튜토 생산 체인 (회귀 대상)

```
철광석 노드 ─[채굴기]─→ iron_ore ─[벨트/출력기/수동]─→ 용광로 ─→ iron
  ─[벨트/출력기/수동]─→ 제작기 ─→ iron_rod / iron_plate
```

| 출력 | 입력 | type | 소요 |
|------|------|------|------|
| `iron_ore` | (없음) | `채굴기` | 30틱 |
| `iron` | `iron_ore`×1 | `용광로` | 15틱 |
| `iron_rod` | `iron`×1 | `제작기` | 5클릭 |
| `iron_plate` | `iron`×1 | `제작기` | 5클릭 |

물류: 출력기(뒤 포트→앞 칸) · 벨트(Factorio형) · 플레이어 수동 운반.

## 5. 폴리시 대상

**물류**: 벨트→출력기→수동 회수 우선순위 · 채굴기 산출 경로 · 용광로·제작기 입력 흡수 · 회수 시 포트·WIP→인벤

**고장·수작업**: 고장 빈도 조정 · 스프라이트 붉게+해결 아이콘 · 제작기·마나 제작기 E키/클릭

**안정성**: 10종+ 배치·회수·세이브 복원 · Console Error 0 (문서화된 경고만)

고장 빈도·수리 클릭 수가 미정이면 `01-mvp-factory-polish_missingReq.md`를 채운다.
