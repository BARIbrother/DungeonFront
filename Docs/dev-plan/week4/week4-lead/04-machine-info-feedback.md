# 기계 정보·피드백

**역할**: Lead · **Week**: 4 · **Issue**: 04  
**선행**: `MachineRecipeUI` · 공장 폴리시

## 1. 이 작업물

기계 선택·정보 패널을 최소 완성한다.  
플레이어가 레시피·WIP·고장을 한눈에 보게 해 뒷산 클리어 중 헤매지 않게 한다.

**코드**: `MachineRecipeUI` 확장 또는 `MachineInfoPanel.cs`

## 2. UI 스펙 (최소)

| 요소 | 내용 |
|------|------|
| 이름 | `displayName` |
| 레시피 | 현재 Recipe 표시명 · 입·출력 |
| WIP | 진행 바 또는 tick/click 잔여 |
| 상태 | Idle / Working / Broken |

레시피 전환: 준비·생산 모두 우클릭 UI. 기계는 현재 선택 레시피만 작업.

고장 연출(월드): 스프라이트 붉게 + 해결 방법 아이콘 — 패널에도 Broken 반영.
