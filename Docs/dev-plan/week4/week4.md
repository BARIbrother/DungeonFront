# Week 4 — 개발 계획 (개요)

**전제**: Week 3 MVP 골격 완료  
**범위**: 뒷산 동굴 (`dungeonId` `001`)만

## 목표

뒷산 동굴 완성 — 튜토리얼부터 스토리 엔딩(대장장이 면허 취득)까지, 한 던전만으로 끊김 없이 플레이 가능하게 콘텐츠·밸런스·폴리시를 채운다.

| 역할 | 작업 |
|------|------|
| Lead | 공장 폴리시·구역 확장·맵/노드·기계 피드백·통합 |
| Dev1 | UI 폴리시·튜토 완성·세이브 안정화·엔딩·통합 |
| Dev2 | 경제 수치 lock·뒷산 의뢰·클리어 라인·상시 의뢰·결산 폴리시 |
| Art | UI 프레임·뒷산 타일·기계 비주얼 러프·전달 |

## 플레이 규모

| 모드 | 일차 | 플레이 시간 |
|------|------|-------------|
| 일반 | 약 40일 | 약 3시간 |
| 스토리 only (시연) | 약 7일 | 약 30분 |

- [ ] 멀티데이 · 게임오버 · 세이브/로드
- [ ] 스토리 엔딩(면허) → Title · Art(프레임·타일) 적용 · 기계 러프
