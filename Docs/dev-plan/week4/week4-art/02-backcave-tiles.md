# 뒷산 동굴 타일

**역할**: Art · **Week**: 4 · **Issue**: 02  
**선행**: Week 1 스타일 가이드

## 1. 이 작업물

뒷산 동굴 Factory용 바닥·맵 경계 타일을 확정한다.  
스타일: **픽셀아트**. 분위기: 평화로운 숲속·초원 (뒷산 동굴).

**경로**: `Assets/Art/Tiles/BackCave/`

## 2. 산출물

| 항목 | 내용 |
|------|------|
| 바닥 | 초원 이미지 **1장** (약 **6400×3200**). Tilemap에서 일부를 나눠 씀. 거의 단색 초원 + 군데군데 흙바닥. 풀은 살포 브러시로 드문드문. 픽셀 경계선만 자연스럽게 — 그 외 디테일은 거의 불필요 |
| 맵 경계 | 숲 타일 **극소수**(현재 기획: **1개**)를 복붙. 반복·빈틈없는 나무 그림. 복붙 전제이므로 퀄리티를 어느 정도 신경 씀 |
| 벽·장식 | 없음 |

Tilemap용 슬라이스 · PPU·피벗은 스타일 가이드 준수.

## 3. 완료 기준

- [ ] 초원 바닥 1장 ≈6400×3200 (단색·흙·살포 풀, 디테일 최소)
- [ ] 반복·빈틈없는 숲 타일 1개 (복붙용, 퀄리티 확보)
- [ ] 뒷산 맵에 적용 가능 · Tilemap 슬라이스
- [ ] PPU·피벗이 스타일 가이드를 준수
- [ ] 튜토~엔딩 시연에서 placeholder 티가 과하지 않음
