## WIP·고장·수리

### 이 작업물

생산 중 **WIP** 보관 + **고장** 발생·클릭 수리. 구 7주 **4주**.

### WIP

- [ ] 레시피 진행 중 입력·중간재 → 기계 WIP 슬롯
- [ ] Production 종료 시 미완 WIP → 재료 환원 (정책 1안 고정)
- [ ] 완성품 → 출력 버퍼 (Week 3 요약에서 인벤 일괄)

### 고장

| 항목 | 내용 |
|------|------|
| 발생 | Production 중 랜덤 (빈도 placeholder OK) |
| 표시 | Sprite 붉은 tint + 아이콘 |
| 수리 | 근접 + 클릭 N회 |
| 효과 | 수리 전 해당 기계 틱 정지 |

### 완료 기준

- [ ] 진행 중 레시피가 WIP에 보임
- [ ] 고장 기계 tint·수리 후 틱 재개
