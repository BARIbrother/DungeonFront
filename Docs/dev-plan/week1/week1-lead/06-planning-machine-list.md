## MVP 기계 10종+ 목록 초안

### 이 작업물

MVP 완료 조건 **「10종 이상의 기계」**를 채우기 위한 **기획 표 v0.1**.  
이번 주는 **이름·역할·수작업·해금 시점**만 확정한다. 레시피 틱·입출력 수치는 후속.

**산출 형식**: 이 레포 `Docs/` MD 표, Notion, 스프레드시트 등 **팀이 보는 곳 1곳**에 공유.

### 표에 넣을 열

| 열 | 설명 |
|----|------|
| # | 1, 2, 3… |
| machineTypeId | `02-data-structure.md` type 문자열 |
| 표시명 (1티어) | UI용 한글명 |
