## 결산 화면 스텁

### 이 작업물

**결산 단계**용 **별도 화면**의 최소 버전 (`00-vision.md`). Week 1에는 의뢰 목록·납품·보상 **없이**, 화면 전환과 **다음 날**만 동작한다.

**씬 이름**: `Settlement` (권장) **또는** Factory 위 **전체 화면 UI 패널**

### 화면에 있을 것

| 요소 | Week 1 동작 |
|------|-------------|
| 제목 텍스트 | `결산` (또는 `1일차 결산`) |
| 일차 표시 | `session.day` (글로벌 HUD와 중복 가능) |
| **다음 날** 버튼 | 클릭 → `AdvanceDay()` + Factory 복귀 |

### 없어도 되는 것 (후속)

- 의뢰 목록, 보유 n / 요구 m, 납품 버튼, 보상 팝업, 미납 처리

### 진입·퇴출

- [ ] `GamePhase`가 `Settlement`가 되면 **이 화면만** 보임 (Factory 숨김 또는 씬 언로드)
- [ ] 「다음 날」 후 `GamePhase` = `Prepare`, [05-scene-flow](./05-scene-flow.md)가 Factory 표시

### 완료 기준

- [ ] Production 종료(또는 디버그) 시 Settlement 화면이 보인다
- [ ] Factory 맵·플레이어는 보이지 않는다
- [ ] 「다음 날」 클릭 시 일차 +1, Factory로 돌아간다
- [ ] 글로벌 HUD 일차가 갱신된다
