## 도트 스타일 가이드

### 이 작업물

**초상화**와 **맵 캐릭터 모션**에 공통으로 쓸 **도트 아트 규칙** 1장.  
팀(Lead·Dev·Art)이 픽셀 크기·팔레트를 맞추는 기준 문서.

**산출물**: PDF 또는 PNG **1페이지** (Figma·PSD 원본 + export)

### 가이드에 반드시 포함할 항목

| 항목 | 작성 내용 |
|------|-----------|
| **픽셀 크기** | 맵에서 캐릭터 **1칸 높이** = 몇 px (예: 32px) |
| **초상 캔버스** | 초상화 PNG 권장 크기 (예: 128×128, 96×96) |
| **팔레트** | 주색 **8~16색** (hex 또는 RGB 목록) |
| **윤곽선** | 있음/없음, 두께 1px 등 |
| **초상 프레이밍** | 머리~가슴 / 어깨 위 — 대화 UI 슬롯 비율 |
| **탑다운 모션** | **4방향** (상·하·좌·우) 기준 |
| **레퍼런스** | 참고 게임·이미지 1~2개 캡처 또는 링크 |

### 팀 합의 (가이드에 기록)

- [ ] Lead [01-grid-map](../week1-lead/01-grid-map.md) **1 unit = 1칸**과 캐릭터 px 높이 대응
- [ ] Dev 대화 UI 슬롯 크기 **미정**이면 가이드에 **임시 규격** 명시 (예: 256×256 슬롯)

### 완료 기준

- [ ] 1페이지 문서에 위 표 항목이 **빈 칸 없이** 채워짐
- [ ] Lead·Dev에게 파일 경로 또는 채널로 **공유됨**
