## 주인공 초상화

### 이 작업물

**대화 UI**·의뢰·스토리에서 쓸 **주인공 얼굴/상반신 초상** 1종.  
맵 위 탑다운 스프라이트([03-protagonist-motions](./03-protagonist-motions.md))와 **같은 인물**로 보여야 한다.

### 캐릭터 설정

- 대장간을 **물려받았으나 정식 면허 없는** 주인공 (`01-core-loop.md`)
- 톤: **밝은 경영 시뮬** — 과도하게 어둡지 않게
- MVP NPC: 이브·레이·네메시스 — **이번 Issue는 주인공만**

### 파일 규격

| 항목 | 권장 |
|------|------|
| 형식 | **PNG**, 알파 채널 |
| 크기 | [01-style-guide](./01-style-guide.md)의 「초상 캔버스」 따름 |
| 여백 | 슬롯에 넣었을 때 잘리지 않게 **상하좌우 패딩** |
| 배경 | 투명 |
| 표정 | **기본 1종** (추가 표정은 후속) |

### 작업 단계

- [ ] [01-style-guide](./01-style-guide.md) 팔레트·윤곽선 적용
- [ ] 정면 또는 3/4 뷰 **초상 1장** 완성
- [ ] export 설정: **픽셀 퍼펙트**, 스케일 없는 1:1
- [ ] 파일명 예: `protagonist_portrait.png`

### 전달

- [ ] **Dev** — 대화 UI 붙일 때 import 경로 (Week 4~ 본격 연동, Week 1은 파일 전달만)
- [ ] 팀 공유 폴더·채널에 업로드

### 완료 기준

- [ ] [03-protagonist-motions](./03-protagonist-motions.md) 탑다운 캐릭터와 **동일인**으로 인식 가능
- [ ] Dev 또는 Lead가 프로젝트 `Assets/Art/` 등에 import 가능한 상태
