## 수작업·수동 운반·회수

### 이 작업물

클릭 **수작업**, 플레이어 **운반**·**수동 회수**. 구 7주 **5~6주**.

### 수작업

- [ ] `requiresManualWork` — 근접 + 클릭 → `manualClickCount` +1
- [ ] 완료 시 출력 버퍼

### 운반·회수

- [ ] 버퍼·벨트 끝에서 **들기** / 다른 입력·인벤에 **놓기**
- [ ] 출력 버퍼 → 클릭/상호작용 → **인벤 즉시**

### Art

- [ ] 수리·작업 모션 Animator 트리거 (있으면)

### 완료 기준

- [ ] 제작기 클릭 N회 완료
- [ ] 채굴기 버퍼 → 용광로 수동 운반
- [ ] 출력 버퍼 인벤 회수
