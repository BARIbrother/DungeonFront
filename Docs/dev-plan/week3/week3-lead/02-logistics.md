## 벨트·출력기 물류

### 이 작업물

**컨베이어 벨트** 배치·이송 + **출력기** 보강. 구 7주 **5주** 물류.

**출력기 기본**은 Week 2 [06-outputter](../../week2/week2-lead/06-outputter.md) (전방향 포트 접근).

### 규격

| 항목 | 내용 |
|------|------|
| 벨트 | 그리드 배치, 방향별 이송 |
| 출력기 | 인접 기계 `inputPort`/`outputPort` **4방향** 중계 (Week 2 구현 확장·다중 연결) |
| 우선순위 | 벨트 > 출력기 > 수동 (1안 고정) |

### 구현

- [ ] 벨트 Prefab·배치·틱 이동
- [ ] 출력기 ↔ 벨트 연동 (출력기 옆 벨트 끝 등)
- [ ] [week2 06-outputter](../../week2/week2-lead/06-outputter.md)와 연결
- [ ] 채굴기 → 벨트/출력기 → 용광로 경로

### 완료 기준

- [ ] 5분 생산 중 벨트·출력기 동작
- [ ] 철 체인이 벨트+출력기로 연결 가능
