# 게임오버·미납 페널티

**역할**: Dev2 · **Week**: 3 · **Issue**: 03  
**선행**: 멀티데이 납기 · W2 납품

## 1. 이 작업물

필수(스토리) 의뢰 실패 → 게임오버 UI → Title.  
일반 의뢰 미납 → **보상 명성의 0.5배**만큼 현재 명성에서 차감.

**코드**: `GameOverController.cs` (`Assets/Scripts/Quest/`)

## 2. 미납 판정 시점

기한이 **다 지난 뒤**에 미납을 판정한다.

예: 1일차 Prepare에 받은 **기한 2일** 의뢰

| 시점 | 상태 |
|------|------|
| 1일차 Prepare ~ 2일차 Settlement | 완료(납품) 가능 |
| 3일차 Prepare | 미납 판정 · 페널티 적용 |

## 3. 판정

| 조건 | 결과 |
|------|------|
| `isMandatory` + 기한 만료 + 미납 | 게임오버 |
| 일반 + 납품 성공 | 정상 보상 |
| 일반 + 미납 (기한 만료) | 명성 − (보상 명성 × 0.5) |

## 4. 게임오버 UI

| 요소 | 내용 |
|------|------|
| 본문 | 「필수 의뢰를 완료하지 못했습니다」 |
| 버튼 | Title 로드 |
| 세이브 | 삭제하지 않음 — 다른 슬롯 재도전 |

## 5. 완료 기준

- [ ] 기한 만료 시점에만 미납 판정 (예: 2일 기한 → 3일차 Prepare)
- [ ] 필수 미납 → 게임오버 → Title
- [ ] 일반 미납 → 보상 명성 ×0.5 차감
- [ ] 필수 성공 시 게임오버 없음
- [ ] 게임오버 후 다른 슬롯 로드 가능
