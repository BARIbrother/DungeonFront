# 경제·명성 해금 UI

**역할**: Dev2 · **Week**: 3 · **Issue**: 02  
**선행**: `GameSessionState.gold/reputation` · Prepare 페이즈

## 1. 이 작업물

Prepare에서 **특정 버튼**을 누르면 펼쳐지는 **기술 해금 트리** UI.  
공장 게임의 일반적인 테크 트리와 비슷하다.

**코드**: `Assets/Scripts/Quest/` — `UnlockManager`, 테크 트리 UI

## 2. UI

- Prepare에서만 버튼·패널 사용
- 버튼 클릭 → 테크 트리 패널 펼침
- 노드 클릭 → **해금 확인 메시지** → 확인 시 해금

## 3. 테크 트리 규칙

| 항목 | 내용 |
|------|------|
| 트리 수 | **여러 개**의 테크 트리 |
| 배치 | **아래 → 위**로 갈수록 높은 테크 |
| 선행 | 이전(아래) 테크를 해금하지 않으면 **해금 불가** |
| 수치·배치 | 일단 **테스트용 더미** 테크·보상으로 설정 |

## 4. 완료 기준

- [ ] Prepare 버튼으로 테크 트리 패널 펼침
- [ ] 테크 트리 여러 개 · 아래→위 고티어 배치
- [ ] 선행 미해금 시 해금 불가
- [ ] 노드 클릭 → 해금 확인 메시지 → 해금
- [ ] 더미 테크·보상으로 동작 확인
