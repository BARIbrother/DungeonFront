# 아이템 레벨 규칙
# 일부 핵심 재료와 장비에는 레벨이 존재한다.
# lv n+1 재료는 동일한 lv n 재료 5개를 합쳐 만들 수 있다.
# 레벨이 높은 재료는 낮은 레벨 재료 대신 사용할 수 있다.
# 단, 자동 생산에서는 기본적으로 레시피 요구 레벨과 같은 재료를 우선 사용하며,
# 상위 레벨 대체 사용은 플레이어가 허용한 경우에만 사용한다.
#
# 생산품 레벨은 레시피에 들어가는 '레벨이 존재하는 재료' 중 가장 레벨을 따른다.
# 보조 재료, 촉매, 마나, 스크롤, 인챈트 속성은 기본적으로 생산품 레벨을 결정하지 않는다.
#
# 레시피에 레벨이 명시된 경우, 해당 재료는 그 레벨 이상의 재료만 사용할 수 있다.
# 따라서 모든 주 재료가 lv2 이상을 요구하는 레시피라면 결과물도 최소 lv2 이상이다.

# 2개 이상의 슬롯에 같은 아이템을 요구하는 경우는 없다.
>>>나무

나무 원목 -> 종이x4
>>>철

철 주괴 -> 철 판

철 판x2 + 철 주괴x4 -> 철 투구
철 판x2 + 철 주괴 x 5 -> 철 각반
철 판x2 + 철 주괴x2 -> 철 부츠
철 막대기x2 + 나무 막대기x2 -> 철 검
철 판x4 + 철 막대기x2 + 나무 막대기x2 -> 철 전투 망치

>>>기초 마력석 
마력석 광석 -> 마력석 결정
마력석 결정 + 철 주괴 -> 마력 코어
마력 코어 + 나무 막대기x2 -> 마나 완드
철 검 + 마력 코어x2 -> 마나강 검
철 투구 + 마력 코어x2 -> 마나강 투구
철 흉갑 + 마력 코어x4 -> 마나강 흉갑
철 각반 + 마력 코어x3 -> 마나강 각반
철 부츠 + 마력 코어x2 -> 마나강 부츠
# ↓ 속성 부여는 레시피가 아니라 인챈트 시스템에서 스크롤을 소모해 처리 (검/투구/흉갑/각반/부츠 공용)
>>>마력 추출
하급 몬스터 마나 정수 -> 마나x10
중급 몬스터 마나 정수 -> 마나x50    
상급 몬스터 마나 점수 -> 마나x100
던전의 주인의 정수 -> 마나x500

>>>마법 스크롤
# 스크롤은 마법 관련 장비에 인챈트하는데 쓰임. 스크롤의 종류도 인챈트로 처리.
종이 + 마나x100 -> 빈 마법 스크롤
빈 마법 스크롤 + 마나x300 -> 원소 스크롤
# 원소는 인챈트 개념. lv1 원소는 화염, 바람, 대지, 물 중 선택
원소 스크롤 + 마나x100 -> 원소 형태 스크롤
# 형태도 인챈트 개념. 공격, 방어. 예 - 화염 방어 스크롤, 대지 방어 스크롤, 바람 공격 스크롤 등
빈 마법 스크롤 + 마력 코어 + 마나x100 -> 빈 2단계 마법 스크롤
빈 2단계 마법 스크롤 + 마나x500 -> 2단계 원소 스크롤
# 2단계는 폭발(화염+바람), 용암(화염+대지), 독(화염+물), 자연(바람+대지), 번개(바람+물), 얼음(대지+물) . 이는 조합식이 아니라 컨셉트
2단계 원소 스크롤 + 마나x250 -> 2단계 원소 형태 스크롤
# 2단계 원소 + 형태

>>>의식
빈 2단계 마법 스크롤 + 마나x1000 -> 의식 스크롤
의식 스크롤 + 철 검x10 + 흑강 검x10 + 백강 검x10 + 진강 검x10 -> 검의 의식
검의 의식x10 -> 전쟁의 의식
의식용 철제 대검 + 검의 의식 -> 집행자의 대검
집행자의 대검 + 전쟁의 의식 -> 전쟁에 물든 집행자의 대검

>>>칠흑석

철 전투 망치 + 백강 판x4 + 백강 막대기x2 -> 백강 전투 망치

>>>고급 마법
마나강 흉갑 + 마나강 각반 + 마나x1500 -> 마술사의 로브
# ↓ 로브도 마찬가지로 속성 부여는 인챈트 시스템 처리 (레시피 아님)

마력 코어 lv2 + 칠흑석 주괴 lv2 -> 흑마법 코어

마나 완드 + 백마법 코어 -> 백마술 지팡이

>>>진강
