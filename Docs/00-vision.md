# DungeonFront

> 상태: **MVP 확정** — 상세 플레이·콘텐츠는 `01-core-loop.md`에서 정리

## 한 줄 정의

던전 입구 대장간에서 공장을 운영하며, 모험가 의뢰에 맞춰 장비를 제작·납품하는 2D 공장 게임.

## MVP 범위

**첫 플레이 가능 버전**에 포함하는 것:

| 포함 | 내용 |
|------|------|
| 던전 | **뒷산 동굴** 1개 (기믹 없음, 튜토리얼·입문) |
| 하루 구조 | **준비 → 생산(5분) → 결산** 3단계 |
| 의뢰 | 수락·납기·납품·실패 페널티 |
| 경제 | **골드**·**명성** |
| 생산 | **기초 기계**·**기초 레시피**·**Factorio형 벨트**·물류(출력기·수동 운반) |
| 스토리 | **스토리 엔딩** (MVP 클리어) · 튜토리얼 · 주요 NPC |

**MVP에서 제외** (추후 — `99-further_implementation.md`):

- 던전 복수화·크로스 던전 (6종 던전, 특수 기믹)
- 엔드 이후 **추가 엔드 컨텐츠**·확장 스토리

## 세계관·설정

- **장소**: 던전 앞 대장간
- **플레이어 역할**: 대장간(공장) 경영자
- **고객**: 던전에 입장하는 모험가들
- **던전**: 배경·분위기용. 던전 자체를 플레이하지 않음 (MVP: **뒷산 동굴**만)
- **톤**: **밝은 경영 시뮬** — 부담 없이 공장을 키워 가는 느낌

## 시점·조작

- **준비·생산**: 탑다운 뷰
- **결산**: **별도 화면** (의뢰 확인·납품·보상 UI)
- 준비: 탑다운에서 공장 배치·확장·의뢰 UI
- 생산: 캐릭터 이동 + 마우스 클릭 (수작업·수리)

## 하루 구조 (3단계)

```
[준비] → [생산 5분] → [결산] → 다음 날
```

### 1. 준비 단계

- **탑다운**에서 공장을 보며 배치·확장
- 그날 **의뢰 수락** (최대 3개, 0개 가능)
- **공장 구조 변경**: **준비 단계**에서 인벤토리의 기계를 **자유롭게 배치·회수**
- **새 구역 구매** (확장)
- **명성으로 기계·레시피 해금**(수동) → **골드로 구매·제작**
- 기타 사전 준비 행동

### 2. 생산 단계

- **5분** 동안 공장 가동
- **탑다운 뷰** — 플레이어 캐릭터를 직접 조작하며 현장 대응
- **수작업 기계** (초반·특수 한정):
  - **기초 기계**: **클릭**으로 가공 단계 진행, **완성품 수동 회수**
  - **특수 컨셉 기계**: 해당 기계에 맞는 수작업 행동
  - 고급 자동화로 갈수록 수작업 비중 감소
- **고장**: 가끔 기계가 고장 → **마우스 클릭**으로 수리
- **물류** (생산 단계 중):
  - **Factorio형 벨트**로 기계 간 이송
  - **출력기**: 별도 기계. 뒤쪽 기계 **출력 포트**의 아이템을 앞 칸으로 운반 (점유 크기·종류별 상이)
  - 출력기 없으면 플레이어 **수동 운반**·**수동 회수**
  - 생산 단계 **도중** 수동 회수 가능. **종료 시** 맵·기계에 남은 완성품은 **인벤토리로 일괄 이동**
- Factorio형 자동화 + **액션·현장 관리**가 결합된 구간
- **생산 단계 종료 시 (기계 리셋)**:
  - 기계 안 **진행 중(WIP)인 제작** → 재료로 환원
  - 기계 **제작 상태 초기화**
  - **완성품** → **인벤토리**로 이동 (보관·누적)

### 3. 결산 단계

- **별도 화면**으로 전환
- 의뢰 **물품 제출 버튼** 클릭 → **인벤토리에서 자동 차감** 후 납품
- 조건 충족 시 보상(골드·명성) 수령
- **며칠짜리 의뢰**: 완성품을 인벤토리에 모아두었다가 납기일 결산에서 제출
- 다음 날로 진행

## 핵심 루프

1. 준비: 의뢰 선택·공장 정비·확장·해금
2. 생산: 5분간 라인 가동 + 수작업·고장 대응
3. 결산: 납품·보상 → 1번으로

## 의뢰 시스템

### 수락 규칙

- **하루 최대 3개**까지 수락 가능
- **0개** 수락도 가능 — 단, **필수(스토리) 의뢰는 반드시 수락 클릭** 필요
- **필수 의뢰** 존재 — 스토리·진행상 반드시 완료해야 하는 의뢰
- 의뢰 **등장**: 일차·조건 **혼합** (고정 + 풀). **누적 명성**이 의뢰별 threshold 이상이면 출현 풀에 추가

### 의뢰 형태

- **(요구 물품, 개수)** 엔트리가 **n개**로 구성
- 예: 철검 ×3, 가죽 갑옷 ×1, …

### 납기

- **대부분 1일** (당일 결산 시 납품)
- **일부 여러 날** 납기 의뢰 존재

### 실패·페널티

| 상황 | 결과 |
|------|------|
| 일반 의뢰 기한 미달 | **명성 차감** — 원래 보상 명성의 **50%** |
| **필수 의뢰** 실패 | **게임오버** → **다른 세이브**로 재도전 (일차·필수 클리어 시 **autosave**) |

## 경제·성장

| 자원 | 용도 |
|------|------|
| **골드** | 재료·**기계**·구역 **구매** (및 일부 **제작** 비용) |
| **명성** | **기계·레시피 해금**(수동, 구매·제작 **가능 상태**로 만듦), **의뢰 출현 풀** 해금(누적 명성·자동) |

## 기계·테크

- 각 기계는 **작업 가능한 레시피가 정해져** 있음 (기계 종류 = 담당 가공)
- **기계 티어**가 높을수록 **더 높은 테크 레벨의 레시피** 작업 가능
- **명성**으로 기계·레시피 **해금** → **골드로 구매** 또는 **제작**으로 획득
- 상위 **티어**도 해금 후 골드 구매·제작 (기존 기계 자동 교체 아님)
- 성장 축: 명성 ↑ → 해금 → 골드·라인 확장 → 상위 레시피·의뢰 대응

## 자원·생산

- **구역 확장** 시 공장 안에 **자원 노드** 배치
- 재료 부족 시 **골드로 일시 구매** 가능
- 생산 라인: Factorio형 (**벨트**·기계·출력기·가공 체인)
- **새 게임**: 시작 기계(채굴기·용광로·제작기·창고)는 **인벤토리**에만 존재. **골드 0 · 명성 0**. 플레이어는 맵 **중앙** 스폰
- **배치**: 준비 단계 **배치 모드** — 하단에 인벤 기계 목록 → 선택 → 맵 클릭 배치. **회전 없음**. 입·출력 포트 방향은 자유. 기계 **점유 크기는 종류별 상이**
- **배치 불가**: 타 기계와 겹침, 허용 구역 밖, 이동 불가 타일. **채굴기**는 **자원 노드 위 1대만**
- **인벤토리**:
  - **용량 무제한**
  - 모든 **창고 오브젝트가 인벤토리 공유** (별도 창고 관리 부담 없음)
  - **기계**도 인벤토리에 보관 가능 → 준비 단계에서 배치·회수
  - 완성품 보관·의뢰 납품·며칠치 누적의 기준
  - **진입 장벽 완화**를 위한 설계 (물류·보관에 신경 쓰지 않고 생산·의뢰에 집중)

## 승리·종료

- **승리**: **스토리 엔딩** (MVP에 구현된 **모든 스토리 의뢰** 완료)
- **게임 규모**: 일반 플레이 약 **40일·3시간** / **스토리 only** 모드 약 **7일·30분** (시연용)
- **엔드 이후 컨텐츠**: 추후 추가 예정

## 플레이어 판타지

- 대장간 주인으로 **설계(준비)** 와 **현장 돌진(생산)** 을 번갈아 하는 바쁜 하루
- 라인을 짜두고도 5분간 **직접 뛰어다니며** 공장을 살리는 긴장감
- 의뢰·확장·명성으로 **점점 커지는 대장간** 경영

## 핵심 재미

- Factorio형 **자동화·생산 라인 설계** (준비 단계)
- 5분 **타임 어택** 생산 + 초반 수작업·고장 대응 (생산 단계)
- **초반 수작업 → 후반 자동화** 성장 곡선
- 의뢰 기반 **목표 제작**·**리스크 관리** (받을지, 몇 개까지)
- 골드·명성 **경영·성장**
- 필수 의뢰 **스토리 압박**

## 설계 방향

- **진입 장벽 완화**: 공유·무제한 인벤, 창고 관리 최소화
- **수작업은 초반·특수에만**: 기본 플레이는 자동화 라인 설계가 중심

## 참고 게임

- **Factorio**: 벨트·기계 자동화, 생산 라인 구축
- *(수작업·고장 대응 등 차별점은 자체 요소)*

## 하지 않을 것

- 던전 탐험·전투 플레이 (던전은 배경만)
- 무제한 의뢰 수락 (하루 3개 상한)
- 필수 의뢰 실패 시 그냥 넘어가기 (해당 일 재시도)

## 메모 (기획 대화 기록)

- 하루 = 준비(턴) + 생산(5분 실시간) + 결산(납품)
- 제작 리셋 = **생산 단계·기계** 한정 (WIP → 재료). 완성품은 인벤토리 유지
- 납품 = 결산 UI 제출 버튼 → 인벤 자동 차감
- 생산 = 탑다운, 초반 클릭 가공·수동 회수 + 클릭 수리
- 인벤 = 무제한·전 창고 공유 (진입장벽 완화)
- 자원 = 구역별 노드 + 골드 일시 구매
- 필수 의뢰 실패 = 게임오버, 다른 세이브로 재도전 (autosave 활용)
- 해금(명성) ≠ 구매(골드)
- 생산 중 수동 회수·운반 / 생산 종료 시 인벤 일괄
- 벨트·출력기 = MVP 포함
- 톤 = 밝은 경영 시뮬
- 결산 = 별도 화면
