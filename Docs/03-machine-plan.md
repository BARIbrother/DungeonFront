### 열 작성 규칙

| 열 | 설명 |
|----|------|
| # | 1, 2, 3… (type마다 1행) |
| machineTypeId | [02-data-structure.md](../02-data-structure.md) type 문자열. 한글 허용 |
| 표시명 (1티어) | UI용 한글명. 최저 티어 이름 = type과 같거나 type을 내포 |
| machineDefId (1티어) | `{machineTypeId}_1` 형식 (`machineTypeId`와 동일한 띄어쓰기) |
| 역할 | 한 줄 (채취, 제련, 가공, 물류, 저장…) |
| 수작업 | `O` = 클릭 가공 / `X` = 틱 자동 |
| 해금 (초안) | `시작`, `명성 N`, `의뢰 00100001` 등 |
| 메모 | 점유 크기·특수 규칙 등 (선택) |

## 시작 지급·기초 체인 (본 표 기준)

### 새 게임 인벤 지급

| machineDefId | type | 표시명 (1티어) | 역할 | 수작업 |
|--------------|------|----------------|------|--------|
| `채굴기_1` | 채굴기 | 채굴기 | 자원 채취 | X |
| `용광로_1` | 용광로 | 용광로 | 광석 제련 | X |
| `제작기_1` | 제작기 | 수동 제작대 | 물리적 형태 변화 제작 | O |
| `창고_1` | 창고 | 창고 | 생산품 저장 | O |

### 생산 체인 참고 (레시피 상세는 후속)

**물리**

```
철광석 노드 ─[채굴기]─→ 철 광석 ─[벨트/출력기/수동]─→ 용광로 ─→ 철 ─[벨트/출력기/수동]─→ 제작기 ─→ 철 막대 / 철 판
                                                                                              └→ 창고
```

**마법** *(2번째 스토리 의뢰 이후)*

```
(마나 원천) ─[마나 추출기]─→ 마나 ─[벨트/출력기/수동]─→ 마나 제작기 / 마법 부여대
                                                              └→ 마나 저장소
(명성 350) 제단 — 의식 카테고리 마법 부여
(명성 500) 주조소 — 거대 카테고리 물리 제작
```
